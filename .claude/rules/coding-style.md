# Coding Style Rules

- Vanilla JavaScript (ES6+), no framework
- ES Modules (`import`/`export`) only
- Functional programming: pure functions, immutability preferred
- `const` by default, `let` only when reassignment needed, never `var`
- No `console.log` in committed code

- No synchronous FileReader
- No direct `node_modules` modification
