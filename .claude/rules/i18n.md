# i18n Rules

- Default locale: `ko` (Korean) — for test compatibility
- Supported locales: `ko`, `en`
- Translation files: `src/i18n/ko.js`, `src/i18n/en.js`
- Use `t('key')` function for all UI strings
- `try/catch` around `localStorage` access — must be Worker/Node-safe
- New UI text: add translation key to both ko.js and en.js
